請將您的圖片放在這個資料夾中

使用說明:
1. 將您喜歡的圖片複製到這個 images 資料夾
2. 圖片可以命名為: 1.jpg, 2.jpg, 3.jpg, 4.jpg, 5.jpg, 6.jpg 等等
3. 支援的格式: .jpg, .jpeg, .png, .gif, .webp
4. 建議圖片尺寸: 800x600 或類似比例
5. 如果要新增更多圖片,請同時修改 game.js 中的 IMAGES 陣列

範例:
- 將第一張圖片命名為 1.jpg
- 將第二張圖片命名為 2.jpg
- 依此類推...

注意:
- 圖片檔名要和 game.js 中的設定一致
- 如果圖片無法顯示,請檢查檔名和路徑是否正確
